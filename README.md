# 윤지호 · Yoon Ji-ho — Web CV

문제 해결형 현장 실무자 · 수출 / 해외영업 인터랙티브 웹 이력서.

- 네이비 + 앰버 "신뢰 · 현장형 전문가" 톤 · Space Grotesk + Pretendard · 한·영 병기
- 고정 nav, 풀스크린 히어로, 값(value) 스트립, 경력 타임라인, 강점 카드, 일하는 방식, 스크롤 리빌
- 의존성 없는 단일 `index.html`

## 로컬 미리보기
```bash
python3 -m http.server 4182   # http://localhost:4182
```
